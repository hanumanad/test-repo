# Creation of 2 Instance are done from AWS console instead of writing any Terraform script (Since it was optional)

OS Version : Ubuntu Server 16.04 LTS
Instance Type : t2.micro


Hostname is configured by executing the below command on both the instances
 # hostnamectl set-hostname msr-test-instance-1.ec2.internal
 # hostnamectl set-hostname msr-test-instance-2.ec2.internal
